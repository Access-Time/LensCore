Getting Started

Welcome to LensCore! This guide will help you get started with our open-source accessibility testing and web crawling platform. Whether you prefer Docker or Node.js, we've got you covered.

What is LensCore?

LensCore is a comprehensive platform for automated accessibility testing and web crawling. It provides:

  • ✓ WCAG Compliance Testing - Automated testing using axe-core
  • ✓ Web Crawling - Configurable depth and URL limits
  • ✓ AI-Powered Analysis - Optional AI insights (OpenAI & Anthropic)
  • ✓ Flexible Storage - Local, S3, or Google Cloud Storage
  • ✓ RESTful API - Easy integration with your workflow

Installation

1. Clone the Repository

git clone https://github.com/Access-Time/LensCore.git
cd LensCore

2. Set Up Environment Variables

Copy the example environment file and configure it: 

cp env.example .env

3. Install Dependencies

make install
# or
npm install

Quick Start

Choose your preferred method to run LensCore:

RECOMMENDED

Option 1: Docker

The easiest way to get started. Everything is pre-configured:

Build and start:

make build-docker

Verify it's running:

curl http://localhost:3001/api/health

Stop the service:

make down

Option 2: Node.js

Run directly with Node.js for development:

Development mode (with hot reload): 

make dev
# or
npm run dev

Production mode:

make build
make start
# or
npm run build && npm start

Make Your First API Call

Test accessibility of a website: 

curl -X POST http://localhost:3001/api/test \
  -H "Content-Type: application/json" \
  -d '{"url": "https://example.com"}'

Configuration

Customize LensCore by editing your .env file:

Core Settings

# Server Configuration
PORT=3001
NODE_ENV=development

Storage

Choose your storage backend: local, s3, or gcs 

STORAGE_TYPE=local
STORAGE_LOCAL_PATH=./storage

Crawling

CRAWL_CONCURRENCY=5
CRAWL_MAX_DEPTH=3
CRAWL_MAX_URLS=50
CRAWL_TIMEOUT=30000

Accessibility Testing

AXE_CONCURRENCY=5
AXE_TIMEOUT=30000

AI Features (Optional)

Enable AI-powered analysis and recommendations: 

AI_ENABLED=true
OPENAI_API_KEY=your_openai_key
# or
ANTHROPIC_API_KEY=your_anthropic_key
